# 分片公链 NEAR

NEAR 是一条使用了分片技术和 PoS 共识机制的底层公链。NEAR 的技术旨在解决“区块链三难困境”（安全 · 可扩展性 · 去中心化）。实际上，以太坊 2.0 与 NEAR 都在使用分片。V 神：NEAR 做得非常不错。

## NEAR 协议有什么特别之处？

- NEAR 的速度非常快，它的技术意味着它可以实现比以太坊低 10000 倍的交易费用。
- 对于区块链小白也十分友好易用。
- NEAR 为开发人员提供了一系列模块化组件，以帮助他们快速跟进开发并快速启动项目。
