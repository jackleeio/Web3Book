# Polygon

[Polygon](https://polygon.technology/)，前身为 Matic Network，本是侧链解决方案，后来重新定位为一个二层扩容方案的聚合器，用于连接和构建与以太坊兼容的区块链网络。

## Polygon 的目标

Polygon 目标是围绕以太坊打造一个像多边形一样的网络。  
简言之，就是以太坊的区块链互联网。

## Polygon 实际上是如何工作的？

Polygon 的新区块链是作为一个“提交链”而非“侧链”构建的。  
在 Polygon 生态系统中，可以构建两种不同类型的链：独立链和安全链。  
独立链可以有自己的共识模型，因此不如使用以太坊共识模型的网络安全，但更灵活。

### Polygon Edge

Polygon Edge 使开发者能够运行具有可定制功能的自己的区块链网络，提供以太坊兼容性。通过 Edge，开发者可以快速启动并运行自己的区块链，支持各种定制功能以满足不同的需求。

### Polygon PoS

Polygon 提供了基于权益证明（PoS）的网络，这使得它能够更高效和低成本地处理交易。同时，它还保留了与以太坊的紧密集成，确保了良好的互操作性。

### 其他功能  

- **高可扩展性**：Polygon 旨在提高以太坊的可扩展性，解决其现有的吞吐量问题。
- **低交易费用**：通过优化的架构，Polygon 提供了低廉的交易费用。
- **广泛的应用支持**：Polygon 支持各种去中心化应用（dApps），从 DeFi 到 NFT，应有尽有。

## V 神观点

V 神：Polygon 的缺点是更重视功能而不是安全。尽管如此，Polygon 的创新和改进确实为以太坊生态系统提供了重要的扩展解决方案。

## 结论

Polygon 作为一个致力于提升以太坊扩展性的解决方案，通过其提交链架构、灵活的链类型和丰富的开发工具，提供了一个强大的区块链互联网平台。其高效、低成本的特点使其成为开发者和用户的理想选择。
