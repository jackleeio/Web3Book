# Celo 链：多稳定币共存

该项目由 a16z 支持。

Celo 是一个基于区块链的去中心化稳定币与金融应用平台。目标是建立一套开放的金融系统，让世界上没有银行账号却有智能手机的人都能容易地获得金融服务。

Celo 正在构建带有内置稳定币系统的移动优先的智能合约平台，以应对新兴经济体中的金融兼容性。

## Celo 协议有两种资产

- 原生代币 CELO（原名 Celo Gold）
- 用于支付的稳定币 Celo USD（cUSD）和 Celo Euro（cEUR）
