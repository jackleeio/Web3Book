# “数字联邦” Tezos

Tezos 是一个对标以太坊的高性能底层公链，亮点是自我修复功能，XTZ 代币的持有者可以自己抵押代币做节点。

## Tezos 的特色

- 采用 PoS 共识机制的智能合约平台，却有很多差异化功能，比如形式验证要容易得多，这使用户和开发者都避免了昂贵的交易。
- 支持数字资产和应用，同时也支持利益持有者通过形式验证过程对其进行自我升级。
- 是整个生态系统中最节能的区块链之一，不及以太坊每年消耗的 0.001%。

## 被 NFT 拯救的公链

基于 Tezos 区块链的 NFT 平台 ONEOf 与格莱美签署了一项为期三年的合作协议。该平台将为第 64、65 和 66 届格莱美奖发布 NFT。

知名游戏行业巨头 Ubisoft 宣布推出支持 NFT 的游戏平台 Ubisoft Quartz，该平台基于 Tezos 区块链搭建。
