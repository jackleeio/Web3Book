# Polygon

前身为 Matic Network，本是侧链解决方案，后来重新定位一个二层扩容方案的聚合器，用于连接和构建与以太坊兼容的区块链网络。

## Polygon 的目标

Polygon 目标是围绕以太坊打造一个像多边形一样的网络。  
简言之，就是以太坊的区块链互联网。

## Polygon 实际上是如何工作的？

Polygon 的新区块链是作为一个“提交链”而非“侧链”构建的。  
在 Polygon 生态系统中，可以构建两种不同类型的链：独立链和安全链。  
独立链可以有自己的共识模型，因此不如使用以太坊共识模型的网络安全，但更灵活。

Polygon Edge 使开发者能够运行具有可定制功能的自己的区块链网络，提供以太坊兼容性。

## V 神观点
