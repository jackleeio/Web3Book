# Cosmos

## 本质上 Cosmos 是一个发链与跨链协议

- 万链互联
- 一键发链

## Cosmos 的最大卖点

- **互操作性**：支持区块链跨链去中心化的交易。
- **可扩展性**：基于 Tendermint 引擎可实现每秒数千笔交易。

## Cosmos vs Polkadot

- **跨链技术最好的 2 个项目**
- **生态哲学**：Cosmos 生态的设计更加开放，Polkadot 生态的设计相对封闭。
- **安全性**：Polkadot 生态整体来看更安全，Cosmos 生态中各链的安全性参差不齐。

## Cosmos 生态发展现状

- 仅次于以太坊生态。
- Terra（LUNA）是 Cosmos 生态中第一名。

## Cosmos 的未来

- 如果以太坊升级迟迟不能到来，那么可能以太坊的领先地位会被 Cosmos 一类的革命者大幅削弱。
- 如果以太坊解决了性能和 gas fee 过高的问题，市场可能会形成一超多强的局面。
- 由于没有历史包袱，Cosmos 可以在技术上探索更多的可能性，建立独有的生态结构。
