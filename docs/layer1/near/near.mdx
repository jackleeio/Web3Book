# 分片公链 NEAR

[NEAR](https://near.org/) 是一条使用了分片技术和 PoS 共识机制的底层公链。NEAR 的技术旨在解决“区块链三难困境”（安全 · 可扩展性 · 去中心化）。实际上，以太坊 2.0 与 NEAR 都在使用分片。V 神：NEAR 做得非常不错。

## NEAR 协议有什么特别之处？

- **快速速度**：NEAR 的速度非常快，它的技术意味着它可以实现比以太坊低 10000 倍的交易费用。
- **用户友好**：对于区块链小白也十分友好易用。
- **开发者支持**：NEAR 为开发人员提供了一系列模块化组件，以帮助他们快速跟进开发并快速启动项目。

## 针对 Web 3.0 应用的最大痛点

NEAR 网络中的每个账户都有一个可定制化的账户 ID，与银行转账很相似，用户只需输入用户 ID 及金额即可完成转账，这大大优化了用户转账时的复杂度，也解决了日常转账时可能出现的不同问题。

## Aurora + 彩虹桥带来极致 Layer 2 体验

NEAR 近期推出了 EVM 兼容链 Aurora，为用户以及开发者提供以太坊 Layer 2 体验的虚拟机。得益于 NEAR 的分片扩容技术，Aurora 的整体吞吐量可达到以太坊主网的 10 倍。

Aurora 之外，NEAR 也在近期正式上线了彩虹桥（Rainbow Bridge），由一套智能合约实现的跨链通信协议，实现两个系统的互操作（彩虹桥只支持以太坊及 NEAR 间的跨链）。

## 总结

NEAR 作为一条创新的分片公链，通过其高效的技术和用户友好的设计，解决了区块链三难困境，并为 Web 3.0 应用提供了强有力的支持。其与以太坊的良好互操作性，以及强大的生态系统，使其在未来区块链发展中占据重要位置。
