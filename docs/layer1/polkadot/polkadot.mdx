# Polkadot 平行链

## Polkadot 的平行链功能

- 将区块链相互连接
- 使用户能够使用他们的 Substrate 框架轻松构建区块链
- 托管区块链，处理其安全性和交易
- 将 Polkadot 上的区块链与其他网络（如以太坊和比特币）连接起来

## Polkadot 的平行链网络架构

- **中继链（Relay Chain）**：作为 Polkadot 的主链，中继链为构成 Polkadot 连接所有这些独立的链的基础。
- **平行链（Parachain）**：通过 Polkadot 网络运行的并行链。
- **转接桥（Bridgechain）**：这条链连接不使用 Polkadot 治理协议的其他区块链。

## Polkadot 创始人

Gavin Wood 其实是以太坊前 CTO，撰写了第一版以太坊黄皮书（技术白皮书），定义了以太坊虚拟机 EVM，更发明了 Solidity 语言。  
早在 2015 年，Gavin Wood 就提出了 Web 3.0 的构想。

## Polkadot 的历程

官方目前支持 100 个平行链插槽，一半用于竞拍，30 个用于平行线程，20 个用于系统平行链。

## 插槽竞拍方式：蜡烛拍卖 + 众贷模式

- **蜡烛竞拍**：蜡烛竞拍是 17 世纪盛行的英式拍卖方式，蜡烛熄灭前一刻出价最高的人将赢得拍卖。
- **众贷模式**：波卡平行链插槽拍卖更像是一种锁仓空投形式。
